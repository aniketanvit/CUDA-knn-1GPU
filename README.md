# k-nearest neighbor using CUDA

the program will generate "time.txt" to store the GPU execution time
more details about the algorithm design please read doc/design.pdf

## Compile the program
```shell
	$ make
```
## Execute the program
```shell
	$ chmod +x ./knn.o
	$ ./knn.o <inputfile>
```
